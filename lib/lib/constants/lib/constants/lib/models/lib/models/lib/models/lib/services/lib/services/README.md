# VibeAcademy AI 🎓

A modern AI-powered learning platform for students built with Flutter, Firebase, and OpenAI.

## ✨ Features

- 🔐 **Firebase Authentication** - Secure sign up/login with email & password
- 📚 **6 Course Categories** - Computer Science, IT, Cybersecurity, Business, Tourism, Media & Communication
- 🤖 **AI Tutor** - Powered by OpenAI GPT-3.5 Turbo
- 📊 **Progress Tracking** - Mark lessons as completed and track your learning journey
- 🌙 **Dark Theme UI** - Beautiful dark theme with purple and cyan gradients
- 💾 **Real-time Database** - Cloud Firestore integration
- 📱 **Responsive Design** - Works seamlessly on all screen sizes

## 🚀 Getting Started

### Prerequisites

- Flutter SDK (>=2.19.0)
- Dart SDK
- Firebase Account
- OpenAI API Key

### Installation

1. Clone the repository
```bash
git clone https://github.com/gavingav254/vibe-academy-ai.git
cd vibe-academy-ai
```

2. Install dependencies
```bash
flutter pub get
```

3. Configure Firebase
- Go to Firebase Console
- Download google-services.json (Android)
- Download GoogleService-Info.plist (iOS)
- Update lib/firebase_options.dart

4. Run the app
```bash
flutter run
```
